# icon.party

Work in progress icon conversion/generation web app.

## Supports

### Presets

- Electron
- PWA
- iOS (AppIcon.appiconset)

### Import formats

- JPG
- PNG
- SVG

### Export formats

- PNG
- ICO (Windows)
- ICNS (macOS)
